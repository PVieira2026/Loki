01 — What is Loki

One extension.
Every workflow.

Loki is a custom browser extension engineered to automate complex workflows, inject dynamic UI features, and optimize processes across multiple internal interfaces (NEO moderation, OLX, Standvirtual and Imovirtual Admin Panel) and client-facing platforms (OLX, STV, IMO).

Built on Scout's legacy, it serves as the core automation layer for Trust & Safety operations. Loki seamlessly intercepts and manipulates DOM structures, maps actions to single clicks, and provides a unified interface to streamline daily validation tasks.

05 — Questions

Common questions.

What exactly is Loki — is it a website or an app? +
Loki is a Chrome browser extension — it works invisibly on top of the sites you already use every day (NEO, Admin Panel, Datadome). It adds a floating toolbar, a right-click menu, and a popup hub that only appear when relevant to the current page.
Who built Loki? +
Loki was designed and developed by Pedro Vieira as an evolution of the original Scout PT extension. While preserving its core foundation, Loki introduces a comprehensive suite of features designed to drastically accelerate Trust & Safety operations:
  • Advanced Automation Pipelines: Automates repetitive moderation tasks, automatically groups duplicate accounts, and speeds up manual listing audits.
  • Case & Salesforce Helpers: Custom built-in tools that automatically parse and structure case details directly from Salesforce queues, reducing manual work.
  • Unified IP & Partner ID Tracking: Instantly fetches partner ad IDs from Atlas, maps user IP locations with country flags, links to IP lookup services, and integrates registry queries like SIAC.
  • Embedded Registry Verification: Features direct checks for companion animal registries (SIAC), local lodging licenses (RNAL), and vehicle odometer kilometer records.
Is Loki safe to use? Does it access data it shouldn't? +
Loki only reads page data from platforms it's configured for (NEO, Admin Panel, Datadome). It does not store personal data, send data outside OLX infrastructure, or access anything beyond the active browser tab. All integrations use official and secure OLX endpoints.
What's the difference between Loki, OLX Validator, and Ticket Tracker? +
Loki is the core browser extension that enhances the daily workflow on NEO, Admin Panel, and Datadome. OLX Validator is a standalone web app that monitors and validates listings across critical categories, including Pet registries (SIAC), local accommodation licenses (RNAL), and car kilometer data for compliance and anti-fraud verification. Ticket Tracker is an event ticket scalping monitoring dashboard. All three share Loki's core automation layer.
